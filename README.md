## README ABOUT PARSER MODULE

This module will help you to read YAML/YML/JSON and get associative array with content.
Module have only 1 function - parse, which is get a path to file and get result as array

### Example of using

Input file:

> {
> "host": "hexlet.io",
> "timeout": 50,
> "proxy": "123.234.53.22",
> "follow": false
> }

Result:
>array(3) {
>  ["timeout"]=>
>  int(20)
>  ["verbose"]=>
>  bool(true)
>  ["host"]=>
> string(9) "hexlet.io"
>}
